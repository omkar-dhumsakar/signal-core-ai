# Security Policy

## Supported Versions

Currently, the following versions of Signal Core AI are supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of Signal Core AI seriously. If you discover a security vulnerability, please do NOT report it by creating a public GitHub issue. 

Instead, please send an email to **security@signalcore.ai**.

We will acknowledge receipt of your vulnerability report within 48 hours and strive to send you regular updates about our progress. If you have not received a reply to your email within 48 hours, please reach out to the project maintainers privately.

Please provide the following details in your report:
* A description of the vulnerability.
* Steps to reproduce the issue.
* Potential impact and affected components.
* Any potential mitigations you might have found.
