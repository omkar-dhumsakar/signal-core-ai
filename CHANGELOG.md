## [Unreleased]

### Added

- Community guidelines: `CONTRIBUTING.md`, `SECURITY.md`, `CODE_OF_CONDUCT.md`.

## [1.0.0] - 2026-06-02

### Added
- **Core RL Engine:** Q-Learning and Deep Q-Network (DQN) architecture supporting multi-echelon dark store routing.
- **Backend Services:** FastAPI implementation with mock SAP/Kafka integrations.
- **Frontend App:** Fully-featured Flutter desktop/mobile application for Store Managers.
- **Features:** 
  - Expiry and spoilage risk monitoring
  - Automated procurement generation
  - Budget pacing logic
  - RLHF (Reinforcement Learning from Human Feedback) directive adjustment mechanisms.

### Changed
- Transitioned internal enterprise pilot code to a generic open-source reference implementation.
